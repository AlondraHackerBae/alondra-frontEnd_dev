World Clock App

A responsive web application that displays the current time for multiple cities around the world, updated in real-time using APIs. The app features a clean and modern interface with dynamic clocks for popular cities, and a search functionality to choose between pre-set locations. Real-time data is fetched using the WorldTimeAPI, and JavaScript ensures seamless updates every second. This project showcases API integration, responsive design, and interactive UI development.

An intuitive and interactive AI-powered web app that helps users craft polished and professional text. Weather App

A sleek and responsive weather app that delivers real-time weather updates for any city worldwide. Users can search for cities to view current conditions, including temperature, humidity, wind speed, and a five-day forecast. The app features a clean design with weather-specific icons, and responsive layout for seamless use on any device. Powered by HTML, CSS, JavaScript, and API integration (OpenWeather API).
